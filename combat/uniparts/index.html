🔩 Uniparts & Set Effects // Equipment ▼

Uniparts are equipment pieces that provide stat bonuses and set effects. Equipping a full set of the same type activates the set bonus. Always match the unipart type to the character's recommended set on their individual page.

Lizard-Type
Increases ATK and CRIT Rate. Increases DMG dealt when using Ultimate.
Best For
Chester (TAR) Gen Narumi (FS) Gen Narumi (JSC) Gen Narumi (JSS) Hikari Shinomiya (TV) Isao Shinomiya (PF) Kafka Hibino (TMWBK) Kikoru Shinomiya (GLF) Kikoru Shinomiya (TMBP) Reno Ichikawa (SA) Rin Shinonome (AH) Soshiro Hoshina (UF) Soichiro Hoshina (KOWJ)
Spider-Type
Increases ATK and Effect Hit Rate. Increases DMG dealt to Kaiju with debuffs or status ailments.
Best For
Aoi Kaguragi (TPRS) Iharu Furuhashi (LT) Keiji Itami (CQ) Jura Igarashi (GD) Reno Ichikawa (TCU) Soshiro Hoshina (ATP) Tae Nakanoshima (TKEPL)
Bovine-Type
Increases DMG Taken Reduction and Shield Application Multiplier. Recovers SP at start of battle. Increases DEF when receiving hits (Max 3 stacks).
Best For
Hakua Igarashi (SP) Isao Shinomiya (TUIW) Kota Tachibana (FDFF) Sagan Shinomiya (DDD) Aoi Kaguragi (FD)
Wyvern-Type

Best For
Haruichi Izumo (TCCE) Kafka Hibino (TMWBR) Mina Ashiro (AGH) Ryo Ikaruga (PR) Soshiro Hoshina (BS)
Ant-Type
