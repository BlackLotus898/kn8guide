🔓 What Is Unlocked Potential?

Unlocked Potential is a post-max enhancement system available for certain characters. It adds five additional nodes (UP1 through UP5) that alternate between new Skill Unlocks (upgraded "+" versions of existing skills) and flat Stat Enhancements.

  • UP1, UP3, UP5: Skill Unlocks — upgrade an existing skill to a stronger "+" version with improved DMG or added effects.
  • UP2, UP4: Stat Enhancements — flat stat increases (commonly HP or ATK, varies per character).
  • Each node must be unlocked in order, I → V.
Note: Not every character has Unlocked Potential data available yet. This page is actively being filled in — check individual character pages for confirmed UP nodes.
